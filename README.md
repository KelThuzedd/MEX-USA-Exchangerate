# MEX-USA-Exchangerate
Forecast on the dynamics of the exchange rate to the US dollar for 5 years and determine which properties of the studied series and factors affect the forecast.
  
Analyze the indicated univariate time series, build reasonable central and interval forecasts  
  
Include the ARCH effect in the model, to predict the prices of financial instruments, it is necessary to
take into account the effect of fat tails when building a forecast
  
Conduct structural break analysis
  
Build a task-specific factor model VAR and evaluate how it affects the forecasts
